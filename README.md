## Variables de entorno:

Este sistema necesita de 2 variables de entorno. El API Rest y Bearer Token:

```
VITE_BEARER_TOKEN=[TOKEN]
VITE_URI_BASE=[API]
```

Con solo estos valores ya podemos correr el sistema en tu equipo usando

```
pnpm run dev
```
